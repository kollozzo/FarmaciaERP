# 💊 FarmaciaERP

Sistema ERP para la gestión integral de una cadena de farmacias, orientado a optimizar procesos de ventas, inventarios, compras y finanzas.

---

## 📌 Descripción del Proyecto

El presente proyecto consiste en el desarrollo de un sistema ERP (Enterprise Resource Planning) enfocado en el sector farmacéutico, con el objetivo de centralizar la información y mejorar la eficiencia operativa de una cadena de farmacias.

Este sistema permitirá gestionar procesos clave como:

- Control de inventarios (lotes y fechas de vencimiento)
- Registro de ventas y emisión de comprobantes
- Gestión de compras
- Integración contable y financiera

---

## 🧱 Arquitectura del Sistema

El sistema está desarrollado bajo el enfoque de **Clean Architecture (Arquitectura Limpia)**, lo que permite una alta mantenibilidad, escalabilidad y separación de responsabilidades.

### 🔹 Capas del sistema:

- **Domain**  
  Contiene las entidades y reglas de negocio principales.

- **Application**  
  Contiene la lógica de aplicación y los casos de uso.

- **Infrastructure**  
  Implementa servicios externos como acceso a base de datos.

- **WebAPI**  
  Expone los endpoints para la interacción con el sistema.
